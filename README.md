# 🕹️ GG Bot — Neural Gaming Assistant

**GG Bot** is a high-performance FAQ assistant designed for instant player support. Built with a "Neural Link" cyberpunk aesthetic, it helps gamers troubleshoot lag, optimize hardware, and master competitive strategies.

## 🚀 Key Systems
* **Performance Tech:** Instant solutions for reducing input lag and optimizing FPS.
* **Pro-Player Knowledge:** Built-in guides for FPS aiming, RPG mechanics, and ranked matchmaking.
* **Hardware Scout:** Up-to-date recommendations for PC builds and peripherals.
* **Immersive UI:** High-tech "Scanline" aesthetic with CSS-driven animations.

## 🛠️ Technical Core
* **Frontend:** HTML5 & CSS3 (Custom Variables, Keyframe Animations).
* **Logic:** Vanilla JavaScript State-Machine for keyword-triggered responses.
* **Performance:** Zero-dependency architecture for lightning-fast load times.

## 🕹️ How to Play (Run)
1. Clone this repository.
2. Ensure your background assets are in the root folder.
3. Open `index.html` in any modern browser.
